### git branch
---

**Git branch** - *[команда]* может быть использована для отображения, создания или удаления веток.

---
**Для отображения всех существующих веток в репозитории введите:**
```bash=
git branch
```
---
**Для удаления ветки:**
```bash=
git branch –d <имя-ветки>
```
