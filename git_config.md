## Git config
---

 **Git config** - *[команда]* которая может быть использована для указания
 пользовательских настроек, таких как электронная почта, имя пользователя, 
формат и т.д. 

---

**Пример использования:**

```bash=
git config --global user.email адрес@gmail.com
```
